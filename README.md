# Lucca

## Installation

```bash
$ npm i
$ bower install
```

## Usage
- In browser:
```bash
$ grunt browser
```
- In nw:
```bash
$ grunt dist-xxxx
```
> Depending on your platform, it can be:
>
>`grunt dist-linux / dist-linux32 / dist-win / dist-mac / dist-mac32`

You can set the `dev` parameter of the `config` object to `true` if you want to use fake pictures.
